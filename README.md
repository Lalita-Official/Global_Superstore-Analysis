# Global_Superstore-Analysis
Sales, Profit, Customer and Shipping Analysis using Excel, SQL and Power BI

## Project Overview

This project analyzes the Global Superstore dataset using Excel Power Query, SQL, and Power BI. The objective is to transform raw sales data into meaningful business insights through data cleaning, analysis, and interactive dashboard visualization.

---

## Business Problem

Global Superstore management requires visibility into sales performance, profitability, customer behavior, product performance, and shipping efficiency across different markets. This project provides an interactive dashboard to support data-driven decision-making.

---

## Tools & Technologies Used

* Excel Power Query – Data Cleaning & Transformation
* SQL – Data Analysis
* Power BI – Dashboard Development & Visualization
* DAX – KPI and Business Measure Creation

---

## Data Cleaning Process

The dataset was cleaned and transformed using Excel Power Query.

Steps performed:

* Removed duplicate records
* Checked and handled missing values
* Corrected data types
* Standardized column names
* Created Shipping Days column
* Validated sales and profit values
* Prepared dataset for SQL analysis and Power BI reporting

## Dashboard Features

### Executive Overview

Provides a high-level summary of business performance including:

* Sales Trend Analysis
* Profit Trend Analysis
* Market Performance
* Key Performance Indicators (KPIs)

### Product Performance Dashboard

Provides insights into:

* Category Performance
* Sub-Category Performance
* Top Products by Sales
* Top Products by Profit
* Profit Margin Analysis

### Customer & Shipping Analysis Dashboard

Provides insights into:

* Top Customers by Sales
* Top Customers by Profit
* Loss-Making Customers
* Ship Mode Performance
* Average Shipping Days

---

## Key Insights

* APAC is one of the highest-performing markets.
* Technology is the most profitable category.
* Phones generate significant sales revenue.
* Standard Class contributes the highest sales volume.
* Several customers generate high revenue but lower profitability.
* Shipping performance directly impacts customer experience.
